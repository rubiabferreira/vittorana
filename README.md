#Vittorana#

Para executar o projeto em modo *development*


`bundle install`

`rake db:migrate`

`rake db:seed`

`rails s`

Dica: Se quiser carregar dados para teste faça o seguinte:

`bundle install`

`rake db:migrate`

`rake db:fixtures:load`

`rails s`

Dessa forma serão carregadas as fixtures do projeto.
